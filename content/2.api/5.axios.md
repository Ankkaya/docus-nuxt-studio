# Axios

## 无感刷新 token

无感刷新 token 即接口请求 token 过期后，利用 accessToken 刷新 token，然后用新 token 重新请求调用失败的接口

需要注意的点有两个

1. 当刷新 token 的时候，禁止其他接口重复吊起刷新 token
2. 当 token 刷新完成后，重新调用请求失败的接口，因此需要在 token 刷新完成前，把这些请求暂存起来

```js
// request.js

// 请求队列
let requestList = []
// 是否正在刷新
let isRefreshToken = false

// 创建 axios 实例
const service = axios.create()

// 响应拦截器
service.interceptors.response.use(
  async (res) => {
    let code = res.data.code || 200
    if (code === 401) {
      // 如果没有在刷新 token，则进行刷新
      if (!isRefreshToken) {
        // 设置刷新状态，正在刷新
        isRefreshToken = true
        // 刷新 token
        const refreshToken = await refreshToken()
        // 保存 token 信息到本地
        setToken(refreshToken)
        // 执行已暂存的请求
        requestList.forEach((cb) => cb())
        // 置空暂存的请求队列
        requestList = []
        // 设置刷新状态为 false
        isRefreshToken = false
        // 重新发起当前请求
        return service(res.config)
      } else {
        // 如果正在刷新 token，暂存其他请求
        new Promise((resolve) => {
          requestList.push(() => {
            res.config.headers['Authorization'] = 'Bearer ' + getToken()
            resolve(service(res.config))
          })
        })
      }
    }
  }
)
```

## 下载/上传文件

### 下载文件

指定请求返回类型

```js
// api/test.js
import request from '@/util/request'

testDownloadApi(data) {
  return request('download', data, 'post', {
    responseType: 'blob
  })
}
```

### 上传文件

```js
// api/test.js
import request from '@/util/request'

testUploadApi(data) {
  return request('upload', data, 'post')
}

// views/upload.vue
/**
 * @author Ankkaya
 * @description 上传文件
 * @param {Object} info - 选中文件信息 
 * @returns {Type} 
 */

uploadFile(info) {
  // info 中属性 file 保存着选中文件信息，具体参数根据使用组件返回的结果
  let fileObj = info.file
  let form = new FormData()
  form.append('file', fileObj)
  // 调用上传方法
  testUploadApi(form)
}
```


## Blob 响应处理

封装 http response 拦截器
```js
// request.js

import axios from 'axios'

const service = axios.create({})

service.interceptors.response.use(
  (response) => {
     // 如果请求返回类型是 blob，直接返回结果
     if (response.config.responseType === 'blob') {
      if (response.status === 200) {
        return response
      } else {
        // 请求失败，直接提示错误信息
        return
      }
     }
  }
)
```
